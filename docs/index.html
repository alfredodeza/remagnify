remagnify

A wlroots-compatible Wayland screen magnifier written in Rust

Quick Install

curl -fsSL https://alfredodeza.github.io/remagnify/install | sh

Features

  • Real-time screen magnification with mouse tracking
  • Adjustable zoom level via scroll wheel (0.01x to 1.0x)
  • Two movement modes: cursor and corner
  • Fractional scaling support for HiDPI displays
  • Multi-monitor support
  • Keyboard shortcut: Press Escape to exit
View on GitHub Releases

Alternative Installation Methods

For Rust developers:

# Fast binary install
cargo binstall remagnify

# Or build from source
cargo install remagnify

Quick Start

# Start with recommended settings
remagnify -z 0.2 --exit-delay 500 --size 1200x600

# Add to Hyprland config
bind = SUPER, M, exec, pkill remagnify || remagnify -z 0.2 --exit-delay 500 --size 1200x600

Requirements

A wlroots-based Wayland compositor (Hyprland, Sway, etc.) with the following protocols:

  • wlr-layer-shell-unstable-v1
  • wlr-screencopy-unstable-v1
